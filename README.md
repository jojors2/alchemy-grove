# alchemy-grove

## Status do projeto

Alchemy Grove está em fase inicial de desenvolvimento. O repositório contém a estrutura base do projeto na Unity, arquivos de configuração, organização inicial dos diretórios e materiais relacionados ao planejamento do jogo.

No momento, a equipe está trabalhando na definição e prototipação das mecânicas principais, incluindo exploração, coleta de ingredientes, criação de poções e progressão do jogador. Por esse motivo, ainda existem poucas funcionalidades implementadas e o código do projeto está em construção.

O projeto será atualizado continuamente conforme novas features forem desenvolvidas, testadas e integradas ao jogo.

## Tecnologias
- Unity
- C#
